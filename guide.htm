wMDB Guide

Editing database records

Edit section

In edit section you can edit any movie you want by selecting movie from datagridview. After that you just simply press save changes and new info saves in database.

Adding records in database

Every tab has its own "Add" -button. When clicking button, new window opens and you can start type a new record to the database. In movies tab, you can also add actors for movie once you have created a movie.

Deleting records from database

Like earlier, every tab has its own "Delete" -button. You simply just choose an item from datagridview and press the button - and VOA'LA! Record deleted.
